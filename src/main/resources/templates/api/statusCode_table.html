Http Status Code 통계분석

두개의 Dags Host Server에 분산되어 있는 API log file을 병합해 API의 http 상태 코드에 따라 분석 및 통계 합니다.

200 (Successful responses : OK) ➔ 요청에 대한 응답이 성공적으로 반환 되었습니다.
400 (Client error responses : Bad Request) ➔ 잘못된 문법으로 인하여 서버가 요청을 이해하지 못해 응답 할 수 없습니다.
401 (Client error responses : Unauthorized) ➔ HTTP 표준에서는 '미승인(unauthorized)'를 의미 하지만, 의미상 '비인증(unauthenticated)'를 명시합니다.
500 (Server error responses : Internal Server Error) ➔ 웹 사이트 서버에 오류가 발생하여 요청에 대해 응답 할 수 없습니다.
